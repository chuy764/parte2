COMANDOS DE HTML

DENTRO DE LOS PRINCIPALES COMANDOS O COMANDOS SIMPLES ENCONTRAMOS COMANDOS COMO:

1. Etiquetas básicas:
<.html>: Define el inicio y el fin del documento HTML.
<.head>: Contiene elementos como metadatos y enlaces a scripts.
<.body>: Engloba todo el contenido visible de la página.

2. Etiquetas semánticas:

<.header>: Encabezado de la página.
<.nav>: Sección de navegación.
<.section>: Agrupa contenido relacionado.

3. Etiquetas para texto:
<.p>: Párrafo de texto.
<.h1>: Título principal.
<.a>: Enlace a otra página.

4. Etiquetas para imágenes:
<.img>: Inserta una imagen en la página.
<.figure>: Envuelve una imagen con su descripción.
